# GPSR_NLP

## Dependencies
### how to setup new RASA enviorment 
install rasa via https://rasa.com/docs/rasa/installation/installing-rasa-open-source/

-> make directory
```
mkdir my_rasa
rasa init 
```
### In AlienWare
activate the venv enviornment
``` source ./venv/bin//activate
```
run the rasa
``` rasa run --enable-api
```
